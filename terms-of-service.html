支持

搬瓦工所有的服务都是完全自我管理的。在保证搬瓦工的服务器理想的工作状态的情况下，搬瓦工不管理、也不提供对客户的 VPS 的支持。换句话说，搬瓦工不协助进行安装和配置应用，排除故障，从备份恢复等工作。 - 这些都应该客户自己负责。

客户需要具备基本的管理 Linux 操作系统所需的知识。

备份

搬瓦工的存储系统是完全冗余的，这意味着一个组件的故障不会影响数据的完整性。 但是，搬瓦工不会备份客户的数据，搬瓦工鼓励客户拥有自己的备份解决方案。

公平分享政策 - VPS 服务

除CPU之外，允许使用所有已分配的资源，不受任何其他限制。

搬瓦工允许在短时间内使CPU达到最大。 但是，请确保1小时平均CPU负载不超过以下级别：

Micro 和 Petite 方案：
  • OVZ Micro-64 方案：1 核的 10%
  • OVZ Micro-96 方案：1 核的 13%
  • OVZ Micro-128 方案：1 核的 15%
  • KVM Petite-128 方案：1 核的 15%
常规方案：
  • OVZ 5G 方案：1 核的 20%
  • OVZ/KVM 10G 方案：1 核的 25%
  • OVZ/KVM 20G 方案：1 核的 50%
  • OVZ/KVM 40G 方案：1 核的 75%
  • OVZ/KVM 80G 方案：1 核的 100%
  • OVZ/KVM 160G 方案：1 核的 100% + 第 2 个核的 50%
  • OVZ/KVM 320G 方案：2 核的 100%
  • OVZ/KVM 480G 方案：3 核的 100%
  • OVZ/KVM 640G 方案：4 核的 100%
  • OVZ/KVM 960G 方案：6 核的 100%
  • OVZ/KVM 1280G 方案：8 核的 100%
特价促销方案：
  • OVZ 3G/128M 特价促销方案：1 核的 10%
  • OVZ 5G/256M 特价促销方案：1 核的 14%
  • OVZ 5G/512M 特价促销方案：1 核的 17%
  • OVZ 30G/4G RAM 特价促销方案：1 核的 50%
  • OVZ/KVM V3 10G/512M 特价促销方案：1 核的 17%
  • OVZ/KVM V3 20G/1024M 特价促销方案：1 核的 25%
  • OVZ/KVM V3 40G/2048M 特价促销方案：1 核的 50%
  • BETA KVM 方案：1 核的 20%

当 VPS 使用比上述更长时间的 CPU 时间时，其 CPU 时钟自动被强制限制为相应的值（即，VPS 将不能使用比上面所示更多的 CPU 周期）。 KiwiVM 控制面板的主屏幕上也显示一条通知。

一旦停止高 CPU 负载使用，系统将自动解除对该服务的限制。 CPU 使用率没有其他限制。 允许使用系统可以分配的尽可能多的 CPU；在上述描述的限制之内，搬瓦工不会暂停或限制服务。

可接受的使用政策

首先，搬瓦工想感谢您花时间查看搬瓦工的服务条款，搬瓦工非常感谢您的努力。订购服务时使用优惠码 ireadtheterms8 以获得小额折扣。但请务必继续阅读。

  • 任何类型的群发邮件（包括双向）
  • DoS或其他攻击（传出或传入）
  • 黑客攻击（包括被黑客入侵的情况）
  • 恶意软件/病毒的分发 - 无论有无知识
  • 端口扫描
  • 加密货币的挖掘（挖矿）
  • IRC活动（客户端，服务器，机器人等）
  • 商标或受版权保护的内容/ mp3 /电影/破解软件或指向此类内容，复制品，仿冒品等的链接 - 搬瓦工尊重所有DMCA投诉

  • 嵌套虚拟化（运行Qemu等）
  • 开放代理（由于滥用代理开放）
  • 打开DNS解析器，打开端口映射器，通过UDP协议打开NTP服务器（由于DoS放大攻击）
  • Tor中继和出口节点
  • 成人内容
  • 数据挖掘，抓取
  • KiwiVM API滥用（造成过多，不合理的API调用）
  • 对搬瓦工的员工使用辱骂性的语言，或用相同的问题或问题发送垃圾邮件

如果搬瓦工发现服务违反了这些条款，搬瓦工将暂停服务，并将解释自动发布到KiwiVM控制面板。 一旦客户确认问题并准备好解决问题，客户可以恢复服务。 这在KiwiVM中完成，不涉及支持。

屡犯的行为可能会招致每次事件50美元的管理费用，并暂停同一账户下的所有服务，直到支付这笔费用为止。

来自VPS的DoS攻击或者发送未经请求的电子邮件可能会招致每个事件50美元的管理费，以及暂停同一账户下的所有服务，直到这笔费用被支付。

在同一日历年（自然年、非付款周期年）3次暂停后，VPS将永久暂停，直到明年1月1日。

搬瓦工保留在不提供理由的情况下终止服务的权利。 在这种情况下，搬瓦工将为未使用的服务提供退款。

黑名单

搬瓦工正在积极监控搬瓦工的 IP 资源，搬瓦工在将每个 IP 地址分配给客户的 VPS 之前对其进行测试。 搬瓦工确保每个新的 VPS 都提供一个干净的 IP。 如果分配给 VPS 的任何 IP 被列入黑名单，搬瓦工将无法退款，直到 IP 不在任何黑名单上。

此外，IP 被列入黑名单将使无法使用 KiwiVM 控制面板中的数据中心迁移功能，直到 IP 不再在任何黑名单上。

取消和退款

客户可以随时在结算控制面板中取消服务。

只要满足以下所有条件，客户可以要求全部退款给他们的账户：

  1. 客户的账户信誉良好，没有违反整个服务条款
  2. 没有任何付款在之前或现在有争议或退款
  3. 任何分配给客户服务的IP都不在任何黑名单中
  4. 月度数据传输配额（使用流量）没有超出
  5. 该帐户下的服务（活跃或取消）数量为3或更少
  6. 向该账户支付的所有款项总和为60美元或以下，付款总次数为10或以下
  7. 帐户是在30天或更少天之内创建的
  8. 客户没有使用其他账户下的退款权利

当要求退款时，搬瓦工将执行以下操作：

  1. 搬瓦工将全额退还从创建帐户开始对帐户进行的所有付款
  2. 该账户下的所有服务将立即终止
  3. 所有的数据，快照和备份将被立即和不可逆转地抹去

退款按原始付款方式返回。

退款处理后，账户可以继续下单购买，但它不会有再一次退款的资格。

如何申请退款：https://bandwagonhost.com/knowledgebase/4/Refunds.html
